🔄 How to Transfer Between Devices

  1. 1
    On Device A: Click "Full Backup (JSON)" — this saves a file to your device
  2. 2
    Transfer the file to Device B via email, cloud drive, or USB
  3. 3
    On Device B: Click "Restore Backup (JSON)" and select the file
  4. ✓
    All your data — expenses, goals, income — is restored!
